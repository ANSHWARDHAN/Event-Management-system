Full Stack MERN Event Booking Platform using React, Node.js, Express, and MongoDB. This project demonstrates how to create a production-ready event management and ticket booking system with features like OTP verification, secure authentication, admin dashboard, and booking approval workflow.
Users can browse events, request ticket bookings with secure email OTP verification, and track their booking status through a personal dashboard. The platform also includes a powerful admin panel where organizers can create events, manage booking requests, confirm payments, and monitor analytics such as revenue, confirmed bookings, and pending requests.
This project is perfect for developers who want to learn how to build a real-world full stack MERN application with authentication, role-based access control, and secure booking logic.


Features of the Event management system
1. Admin can create, update , delete , read all the event at the admin dashboard
2. User can register for the event and the then the admin approve the events accordingy
3. OTP is send when user register on the website and also when the user register for the event on the website implemented using the nodemailer
4. Real time updates notification via socket.io so that we can make it live or interactive
5. Data stored on the mongodb so that there is no issue of sql injection in it
